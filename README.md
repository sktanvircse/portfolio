## ✨ Features

- 🎯 **Parallax Scrolling** — multi-layer depth effect on scroll
- 🖱️ **Custom Cursor** — glowing dot with a lagging ring follow effect
- ⌨️ **Typing Animation** — hero text types itself on load
- 📜 **Scroll Progress Bar** — gradient bar tracks reading progress
- 🌊 **Scroll Reveal** — sections fade and slide up as you scroll
- 🔮 **Floating Orbs** — animated blurred color blobs in the hero
- 🧲 **Magnetic Skill Chips** — subtle push effect on hover
- 🃏 **Project Cards** — lift + glow shadow on hover
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- ⚡ **Zero Dependencies** — pure HTML, CSS & JS, no frameworks

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/sktanvircse/portfolio.git
cd portfolio
```

### Run locally

Just open the file in your browser — no build step needed:

```bash
# Option 1: Open directly
open index.html

# Option 2: Use VS Code Live Server extension
# Right-click index.html → "Open with Live Server"
```

## 🌍 Deployment

This site is deployed via **GitHub Pages**.

To deploy your own fork:

1. Go to your repo → **Settings** → **Pages**
2. Set source to `main` branch → `/ (root)`
3. Click **Save**
4. Visit `https://<your-username>.github.io/portfolio`
